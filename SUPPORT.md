# Support

## Community Support

Use GitHub Issues for:
- bug reports
- feature requests
- setup questions

Please include:
- OS and runtime details
- Docker/non-Docker mode
- relevant `.env` settings (without secrets)
- logs and reproduction steps

## Paid Support

Paid support can include:
- environment setup and hardening
- production tuning for long sessions
- recovery/reliability improvements
- custom integration work

Contact the maintainer to discuss scope, timeline, and pricing.

## Response Expectations

- Community support: best effort.
- Paid support: prioritized according to agreed scope.
