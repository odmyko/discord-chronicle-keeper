# Contributing

Thanks for contributing to Discord Chronicle Keeper.

## How to contribute

1. Fork the repository and create a feature branch.
2. Keep changes focused and include tests when possible.
3. Open a Pull Request with a clear description of what changed and why.

## Sign-off (recommended)

Please sign off commits:

```bash
git commit -s -m "Your message"
```

This adds a `Signed-off-by` line to certify you have the right to submit the code.
